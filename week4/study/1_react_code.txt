특정 파일을 불러오는 구문 
브라우저 환경이 아닌 Node.js에서 지원하는 기능 

번들러
번들 = 묶는다
이러한 기능을 부라우저에서도 사용하기 위해서 사용
파일들을 모두 합처서 하나의 파일을 생성, 최적화 과정에서 여러 개의 파일로 분리될 수도 있다

대표적인 번들러 
웹팩(리액트에서 주로 사용), Parcel, browserify

웹팩의 로더
SVG 파일과 CSS 파일을 불러와서 사용
css-loader = CSS 파일 불러오기
file-loader = 웹 폰트, 미디어 파일 불러오기
babel-loader = 최신 자바스크립트 문법을 특정 자바스크립트 버전을 변환
